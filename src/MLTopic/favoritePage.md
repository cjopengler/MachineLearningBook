# 课题三 最个性网络阅读器
我们每天都会被各种新闻，知识，文章等内容环绕，但是，我们真正感兴趣的，其实不多。于是，我们花费了很多时间会逐个的浏览，然后再精读自己喜欢的文章。那么为什么不做一个最个性网络阅读器呢？

## 内容的来源

1. csdn文章和论坛。因为CSDN有开放平台接口
2. 微博
3. 网页。基于爬虫的网页抓取，然后，根据机器学习对网页内容按照自己的兴趣进行分类。然后，自己能够浏览感兴趣的文章。
4. RSS源
